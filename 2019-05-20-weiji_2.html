专业综合实验微机部分（二）

始

继续填坑，今天晚上搞完三次微机实验。第2次时间2019.03.19

实验

内容

  利用8255A可编程并行口和8253计数器两个芯片，完成以下两个内容。

  内容一：利用8255A的PA口作为输入口，采集8个开关的信息，利用8255A的PB口作为输出口，控制8个发光二极管，开关具有延迟效应，延迟效应由计数器工作方式0来控制。使得打开开关时，不立刻亮或灭。

  内容二：设计一个电路，不需要开关，程序控制8个灯循环点亮，每个灯亮大于0.25秒，即一个循环周期大于2秒。

代码

内容一

CODE    SEGMENT
ASSUME  CS:CODE
IOCONPT EQU    0FF2BH
IOBPT   EQU    0FF29H
IOAPT   EQU    0FF28H
IOCPT   EQU    0FF2AH
TCONTRO EQU    0043H
TCNO0   EQU    0040H
ORG     11B0H
START:  MOV    AL,91H
        MOV    DX,IOCONPT
        OUT    DX,AL
        NOP
        NOP
        NOP

IOLED1: MOV    DX,IOAPT
        IN     AL,DX
        MOV    BL,AL
        MOV    DX,TCONTRO
        MOV    AL,30H
        OUT    DX,AL
        MOV    DX,TCON0
        MOV    AL,12H
        OUT    DX,AL
MOV     AL,7AH
OUT     DX,AL
IOLED2: MOV    DX,IOCPT
        IN     AL,DX
        CMP    AL,01H
        JNE    IOLED2
        MOV    AL,BL
        OUT    DX,AL
        JMP    IOLED1
CODE    ENDS
END     START

内容二

CODE        SEGMENT
ASSUME      CS:CODE
ORG         1234H
IOCONPT     EQU    0FF28H
IOBPT       EQU    0FF29H
START:      MOV    DX,0FF2BH
            MOV    AL,90H
            OUT    DX,AL
            NOP
            NOP
            NOP
            MOV    AL,0FEH
D1:         MOV    DX,IOBPT
            OUT    DX,AL
            MOV    CX,0FFFFH
            ROL    AL,1
            CALL   ABCD
ABCD:       LOOP   ABCD
            JMP    D1
            RET
CODDE       ENDS
END         START
