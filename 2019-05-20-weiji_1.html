专业综合实验微机部分（一）

前言

  这已经是大三下学期了，专业综合实验有数电部分和微机部分，数电用FPGA开发板和Verilog语言；微机部分依旧是实验箱和汇编语言。三月份做的实验被我拖到五月份，前几天刚做完TTL交通灯，集成电路和电子器件的实验又马上到来。我抓紧先把微机的坑给填上。

  第一次微机实验在2019年3月12号，名称：8253定时计数实验，分三个内容，以下开始复制粘贴实验报告。

实验内容

1.计数器0工作方式0，即计数结束中断，OUT与灯L1连接，CLK选择连接左下方T0-T7插口中的一个。T0-T7为分频器输出频率，实验箱频率源为8MHz；T0-T7=4MHz，2MHz，1MHz，500KHz，250KHz，62500Hz，31250Hz；从中选一个插口，即选一种频率（频率是指1s内输入的脉冲数）；要求计数器初值等价于时长1s。

2.计数器工作方式1，OUT灯L2连接，GATE连接开关K1，CLK同上设计（可任选一个频率及计数器初值）。观察开关闭合时，灯的亮灭规律。

3.计数器2工作方式2，速率发生器，OUT与灯L3连接；CLK同上设计（可任选一个频率与计数器初值），观察灯的亮灭规律

实验程序

实验内容一

CODE    SEGMENT
ASSUME  CS:CODE
        ORG 1290H
START:  JMP TCONT
TCONTRO EQU 0043H
TCON0   EQU 0040H
TCONT:  MOV DX,TCONTRO
        MOV AL,,30H
        OUT DX,AL
        MOV DX,TCON0
        MOV AL,12H
        OUT DX,AL
        MOV AL,7AH
        OUT DX,AL
        JMP $
CODE    ENDS
END     START

实验内容二

CODE    SEGMENT
ASSUME  CS:CODE
        ORG 1290H
START:  JMP TCONT
TCONTRO EQU 0043H
TCON0   EQU 0041H
TCONT:  MOV DX,TCONTRO
        MOV AL,72H
        OUT DX,AL
        MOV DX,TCON0
        MOV AL,12H
        OUT DX,AL
        MOV AL,7AH
        OUT DX,AL
        JMP $
CODE    ENDS
END     START

实验内容三

CODE    SEGMENT
ASSUME  CS:CODE
        ORG 1290H
START:  JMP TCONT
TCONTRO EQU 0043H
TCON0   EQU 0041H
TCON1   EQU 0040H
TCONT:  MOV DX,TCONTRO
        MOV AL,74H
        OUT DX,AL
        MOV DX,TCON0
        MOV AL,24H
        OUT DX,AL
        MOV AL,0F4H
        OUT DX,AL
        MOV DX,TCONTRO
        MOV AL,32H
        OUT DX,AL
        MOV DX,TCON1
        MOV AL,12H
        OUT DX,AL
        MOV AL,7AH
        OUT DX,AL
        JMP $
CODE    ENDS
END     START
