常用命令
8月 21 2019

前言

整理一下维护博客常用的命令，就不用老是百度来必应去，麻烦死了。首先装上Git与Node.js。新建一个名为Hexo的文件夹，在此文件夹内右键Git Bash Here，然后按顺序执行指令就行。这是给我自己看的，所以写的比较简略。

Hexo

Hexo乃是万恶之源（手动滑稽），先来说它。

$ npm install -g hexo-cli
$ hexo init
$ npm install

初始化完成，修改模板里的Front-matter，添加上标签，分类；修改_config.yml。

$ hexo g
$ hexo s
$ npm install hexo-deployer-git --save
$ hexo d

Git

连接GitHub

安装完成之后，输入命令来完成设置

$ git config --global user.name "你名字"
$ git config --global user.email "你邮箱"

生成SSH，我一般重装系统才会搞这一块的命令，所以旧的肯定没了，用不着检查和备份。

$ ssh-keygen -t rsa -C "GitHub注册邮箱"

在GitHub里粘贴，测试是否成功

$ ssh -T git@github.com

git init
git add .
-git status

git remote add origin 链接
